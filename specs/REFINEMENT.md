# Deployment Platform Recommendation

**Recommended:** Vercel


## Candidates


### Vercel ✓

**Fit:** high

**Reason:** Native home for Next.js, excels at static generation, SSR, and API routes, offering a superb developer experience for this stack.


### Railway ~

**Fit:** medium

**Reason:** A strong option for hosting full-stack applications with databases and backend services, capable of deploying both frontend and backend components.


### GitHub Pages ✗

**Fit:** low

**Reason:** The plan requires server-side rendering for auth-gated dashboards, which Pages cannot serve.
